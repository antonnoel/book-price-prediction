# book-price-prediction
This project analyzes the famous Amazon top 50 bestselling books dataset to explore some of the factors that drive a book's pricing. It uses k-NN, random forest and a linear regression machine learning model to predict book prices based on features like user ratings, review counts, and publication year.
We will break the dataset into a training set (~80%) and into a testing set (~20%). We will use classical statistics (histograms, means, medians etc.) to see what parameters correlate with the price of the book. Then we will train the data with the three models mentioned above. After that we will test the testing set and compare the predicted price with the actual price. We will compare the three models to see which did the most accurate prediction. 
Also we will add some more factors than what is in the raw dataset. We will predict both the price and the adjusted price as well, where we take the annual US inflation into account. ALso we will add a title length parameter to see if it correlates with the priceing of a book.
